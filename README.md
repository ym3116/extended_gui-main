# extended_gui (icds final project)
● Created an online chat system with secure login system with password and user interface
● Embedded a snake game and tic tac toe double-player game with Python socket and Tkinter

To run the chat system:

1. run python3 chat_server.py in your terminal
2. run python3 chat_cmdl_client.py in your terminal
3. You enter the login interface!
